intSteps <- aggregate(activity$steps,

intSteps$interval <-

Part Four

Impute the missing values

Firstly we need to find the number of missing values/ rows with NA

missing_values <- sum(is.na(activity$steps))

To impute the missing values we first create a mask to hold the newly create dataset in, impMissVal

impMissVal <- activity

From this we have choosen to use the Hmisc package in R to impute the missing values, simply choosing the mean value as the imputed figure.

impMissVal$steps <- impute(activity$steps, fun=mean)

As we have previously seen, the first several observations of steps have NA values. Lets have a quick look to see if this is still the case
